Detailed Syllabus

Unit 1

Digital Components and Data Representation: Number Systems, Convenis BCD, Gray Code, Excess) Code, Paray and Hamming Code
Boolean algebra and minimization: Boolean expressions and their minimization using algebraic identities; Karnaugh map representation and minimization


Unit 2

Combinatorial Circuitic Design Procedure: Adder, Subtractor, BCD Adder. Miriplexers, De-mahtiplesers, Encoders and Decoders
Sequential Switching Circuits: Latches, Flip-Flops, Introduction in Register and Counter


Unit 3

Computer Arithmetic: Data representation, Addition and subtraction, multiplication Algorithms, Division Algorithms, Floating-point Arithmetic operations
Basic Computer Organization and Design: Instruction codes, Computer Registers Computer instructions, Timing and Control, Instruction cycle, Memory Reference


Unit 4

Central Processing Unit: General Register Organization. Instruction Formats, Addressing modes, Data Transfer and Manqulation, Program Control
Micro programmed Control: Control memory. Address sequencing, micro p example, design of control unit.


Unit 5

Memory Organization: Memory Hierarchy, Main Memory, Auxilary memory, Associate Memory, Cache Memory, Input-Output Organization: Input-Output Interface, Asynchronous data transfer, Modes of Trander, Priority Interrupt Direct memory Access
